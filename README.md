# cs32-final-project
My CS32 Final Project with Ronak and Rishab

Description: This project connects to Kalshi’s API to fetch live prediction market data. It is one piece of a larger tool we want to create that can identify arbitrage opportunities on Kalshi to guarantee a profit. While this a lofty goal, this tool brings us one step closer to being able to analyze the market using coding. Overall, we are trying to create a clean pipeline that reliably retrieves and organizes Kalshi market data so it can be read and worked with in Python.
